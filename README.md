# Flow

Minimal browser extension for a calmer YouTube and cleaner tab bar.

## Function
- **On supported sites**: Hides home feeds, “For You” style recommendations, sidebars, and notification-style distractions while keeping core actions (like search and playback) usable.
- **Browser**: Closes empty new tabs (e.g. `chrome://newtab`) and takes you back to your last tab, while allowing links that intentionally open in new tabs.

Works on major Chromium-based browsers (Chrome, Edge, Brave, Opera, Arc, Vivaldi, etc.).
